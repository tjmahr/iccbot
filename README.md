# ICC Bot

<!-- badges: start -->

<!-- badges: end -->

ICC Bot is a Shiny app for computing intraclass correlation coefficients
to estimate interrater reliability.

You can install and run the app (locally):

``` r
# remotes::install_github("tjmahr/iccbot")
iccbot::run_app()
```
